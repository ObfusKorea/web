An Obfuscation Tool for C and Binary Programs

OBFUS is an obfuscation tool for software copyright and vulnerability protection. It provides obfuscation for C code and binary programs, so choose what to obfuscate and paste (or upload) your code below. And you can get a obfuscated version of your work. Is there any difficulty using the web page? Look how to Use it!

How to Use OBFUS

  1. First, select the tab according to the level you want to obfuscate.
  2. Paste (or upload) your code.
  3. And select how to obfuscate your work. There is some options you can choose, so select whatever you want :) (The description to the right of the option may help)
  4. Just click "Obfuscate" to apply them.
  5. When obfuscation ends, the "Download" button appears.
  6. Download the obfuscated version of your work and enjoy it!
